# PyBuster 🚀 
**Simple & Fast Directory Scanner for Termux**

أداة بسيطة وسريعة لفحص مسارات المواقع واكتشاف الملفات المخفية، مصممة خصيصاً لتعمل بكفاءة على تطبيق Termux.

---

## 📥 التثبيت (Installation)

افتح Termux وانسخ الأوامر التالية بالترتيب:

```bash
# تحديث المستودعات وتثبيت git
pkg update && pkg upgrade -y
pkg install git python -y

# تحميل الأداة من حسابك

git clone https://github.com/studioexe78-hub/Pybuster
# الدخول للمجلد (انتبه لحالة الأحرف)
cd Pybuster

# تشغيل سكربت التثبيت
bash install.bash
