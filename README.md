# PyBuster 🚀 

**PyBuster** هي أداة بسيطة وسريعة مبرمجة بلغة بايثون للقيام بعملية فحص المسارات (Directory Bruteforcing) للمواقع الإلكترونية، وهي شبيهة بأداة Gobuster الشهيرة ولكنها مخصصة للعمل بكفاءة على **Termux** وأنظمة Linux.

---

## 🛠️ المميزات (Features)
* **سرعة عالية:** تعتمد على نظام الـ Multi-threading لفحص عدة روابط في نفس اللحظة.
* **دعم اللواحق:** يمكنك البحث عن ملفات معينة (مثل `.php` أو `.html`).
* **واجهة ملونة:** نتائج واضحة (الأخضر للمسارات الموجودة، الأحمر للمحمية).
* **سهولة التثبيت:** تعمل بأمر واحد فقط بعد الإعداد.

---

## 📥 التثبيت (Installation)

للتحميل والتثبيت على **Termux** أو أي نظام ليدكس، افتح الطرفية واكتب:

```bash
# تحميل الأداة من GitHub
git clone [https://github.com/studioexe78-hub/Pybuster](https://github.com/studioexe78-hub/Pybuster)

# الدخول لمجلد الأداة
cd PyBuster

# تشغيل سكربت التثبيت التلقائي
bash install.sh
